### 快捷键

|                  | 现                         | 原             | 修改地址                           |
| ---------------- | -------------------------- | -------------- | ---------------------------------- |
| 格式化代码       | alt + shift + L            | ctrl + alt + L | settings -> Keymap ->reformat Code |
| 删除一行         | ctrl + F                   |                |                                    |
| 复制一行（选中） | ctrl + D                   |                |                                    |
| 换行光标在上     | ctrl + alt + enter         |                |                                    |
| 换行光标在下     | ctrl + shift + enter       |                |                                    |
| 打标签           | ctrl + shift + 数字（1~9） |                |                                    |
| 快速定位标签     | ctrl + 数字（1~9）         |                |                                    |
| 进入到方法       | ctrl + alt + B(鼠标左键)   |                |                                    |
| 搜索             | ctrl  + shift + F          |                |                                    |
